# Portfolio

Static portfolio website for Chennama Charan Kesava Reddy.

## Publish Online

### GitHub Pages

1. Create a public GitHub repository.
2. Upload all files from this folder.
3. Open repository settings.
4. Go to Pages.
5. Select Deploy from branch.
6. Choose the `main` branch and root folder.

Your website will be available at:

```text
https://your-github-username.github.io/your-repository-name/
```

### Netlify

1. Go to Netlify Drop.
2. Drag this entire folder into the upload area.
3. Netlify will create a public website link.
